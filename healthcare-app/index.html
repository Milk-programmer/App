⚙️ Google Sheets Integration

Setup Instructions:

1. Create a new Google Sheet at sheets.google.com

2. Name it "Hospital Appointments"

3. Add these column headers in row 1: Timestamp, ID, Patient Name, Phone, Email, Patient Type, Service, Date, Time, Shift, Notes, Status

4. Go to Extensions > Apps Script

5. Paste the provided script code and deploy as Web App

6. Copy the Web App URL and paste it below
