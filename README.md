# HealthCare Pro - AI Appointment Assistant

A comprehensive healthcare appointment scheduling application designed for health organizations offering all types of surgical procedures. The system operates 24/7 with two shifts:

- **Morning Shift**: 9:00 AM - 10:00 PM
- **Night Shift**: 10:00 PM - 9:00 AM

## Features

### Surgical Categories
The app supports appointments for 12 major surgical categories:

1. **Cardiac** - Heart Bypass, Valve Replacement, Angioplasty, Pacemaker, Heart Transplant
2. **Orthopedic** - Knee/Hip Replacement, Spine Surgery, Arthroscopy, Fracture Repair
3. **Neurological** - Brain Tumor Removal, Spinal Fusion, Craniotomy, Epilepsy Surgery
4. **General** - Appendectomy, Hernia Repair, Gallbladder Removal, Bowel Resection
5. **Plastic** - Breast Augmentation, Liposuction, Rhinoplasty, Facelift
6. **Urological** - Kidney Stone Removal, Prostate Surgery, Cystoscopy, Vasectomy
7. **Gynecological** - Hysterectomy, C-Section, Ovarian Cyst Removal
8. **ENT** - Tonsillectomy, Sinus Surgery, Ear Tube Insertion, Thyroid Surgery
9. **Ophthalmic** - Cataract Surgery, LASIK, Glaucoma Surgery
10. **Vascular** - Varicose Vein Treatment, Aneurysm Repair, Bypass Surgery
11. **Oncological** - Tumor Removal, Lumpectomy, Whipple Procedure
12. **Emergency** - Trauma Surgery, Emergency Appendectomy, Bleeding Control

### Patient Types
Supports all patient demographics:
- Adult (18-65)
- Pediatric (0-17)
- Senior (65+)
- Pregnant
- Critical Care
- Outpatient
- Inpatient

### Key Features
- 🤖 AI-powered chatbot assistant
- 📅 Date and time selection with 24/7 availability
- 👥 Patient type classification
- 🏥 Multiple surgery categories and procedures
- 📧 Email confirmation
- 📱 SMS confirmation

- 🚨 Emergency appointment handling
- 💬 Quick action buttons for easy navigation

## Installation

```bash
npm install
npm run build
```

## Development

```bash
npm run dev
```

## Usage

1. Open `index.html` in a web browser
2. Use the chat interface to schedule appointments
3. Select from quick action buttons or type your requests
4. Follow the conversation flow to complete booking

- TypeScript
- HTML5/CSS3
- esbuild for bundling
- Font Awesome icons
